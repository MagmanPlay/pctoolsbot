# 🖥️ PCTools: Telegram PC Controller

Мощный, скрытный и автономный инструмент для удаленного управления компьютером (Windows) через Telegram бота. 

Отличительная черта проекта — высочайшая отказоустойчивость сети и работа **без прав администратора**. Бот способен пробивать жесткие корпоративные фаерволы (DPI) и управлять системой даже из ограниченных учетных записей.

> ⚠️ **ДИСКЛЕЙМЕР:** Данный проект создан исключительно в образовательных целях и для личного использования (управления собственными ПК). Автор не несет ответственности за любой ущерб, причиненный использованием данного программного обеспечения. Использование скрытого управления на чужих ПК без согласия владельца незаконно.

## 🔥 Главные фишки

* **Не требует прав Администратора:** 95% функционала (включая работу с файлами, скриншоты, управление процессами, веб-камеру и даже WireGuard) работает из-под обычного пользователя.
* **Тройное резервирование связи (Smart Connect):** Бот абсолютно автономен. При запуске он сам решает, как пробиться к Telegram API: через личное зеркало (Relayer), через скрытый туннель WireGuard или напрямую. 
* **Встроенный WireGuard (User-Space):** Бот умеет поднимать WG-туннель прямо в оперативной памяти (через `wireproxy`), не создавая виртуальных сетевых адаптеров и не требуя прав админа.
* **Smart Update:** Бот умеет скачивать новую версию себя по прямой ссылке (`.exe`), заменять старый файл и надежно перезапускаться "на лету" с помощью скрытой VBS-прослойки.

## 🛠 Возможности

### 👁‍🗨 Наблюдение и Медиа
* **Скриншоты:** Быстрый скриншот экрана или полноразмерный HD-снимок с точной отрисовкой позиции курсора.
* **Веб-камера:** Тихое создание фото. Встроена система *Auto-Repair* — если камера зависла или занята, бот попытается программно перезапустить драйвер захвата без запроса админских прав.
* **Микрофон:** Фоновая запись звука (24kHz). Автоматическая нарезка и отправка файлов, если их размер превышает лимиты Telegram (по 21 МБ).

### 📂 Файлы и Процессы
* Удобное скачивание файлов с ПК и загрузка файлов на ПК прямо через чат.
* Скачивание файлов на ПК по прямой URL-ссылке.
* Просмотр топ-25 процессов по потреблению **ОЗУ** (а не CPU).
* Завершение (Taskkill) любых процессов.

### 🖱 Управление вводом
* Эмуляция движений мыши, кликов и нажатий клавиш клавиатуры (`Alt+F4`, `Win+D`, `Enter` и др.).
* **Менеджер ввода:** Блокировка физической мыши и клавиатуры (отличная защита от детей или котов), инверсия кнопок мыши.

### ⚙️ Дополнительно
* **Выполнение CMD-команд:** Невидимое выполнение команд с возвратом вывода (`stdout`/`stderr`) прямо в чат, либо запуск видимого окна терминала.
* **Управление питанием:** Выключение и перезагрузка компьютера.
* **Агрессивные уведомления:** Вывод системных `MessageBox` сообщений поверх абсолютно всех окон (даже поверх игр), перехватывая фокус на себя.
* **Самоуничтожение:** Полное удаление скрипта из системы и автозагрузки по одной кнопке.

## 🚀 Установка и Настройка

### 1. Подготовка
Убедитесь, что у вас установлен **Python 3.8+**.
Установите необходимые зависимости:
```bash
pip install pyTelegramBotAPI comtypes mouse opencv-python psutil sounddevice numpy scipy Pillow pySmartDL pynput pycaw requests urllib3 pysocks
```
### 2. Настройка конфигурации
Откройте скрипт PCToolsBot.py и измените базовые настройки в начале файла:

```Python
my_id = 123456789  # Ваш Telegram ID (бот будет игнорировать всех остальных)
bot_token = 'ВАШ_ТОКЕН_ОТ_BOTFATHER'
```
### 3. Компиляция в EXE с зашитым WireGuard (Рекомендуется)
Чтобы бот был полностью портативным, мы "вшиваем" в него утилиту wireproxy.exe (она будет распаковываться в фоне при необходимости).
Скачайте wireproxy.exe для Windows amd64 (приложена в релизах) и положите рядом с PCToolsBot.py.

Скомпилируйте бота командой:

```Bash
pip install pyinstaller
pyinstaller -w -F --add-binary "wireproxy.exe;." PCToolsBot.py
```
Значения флагов:

-w — скрывает черное окно консоли (работает абсолютно незаметно).
-F — упаковывает всё в один цельный .exe файл.
--add-binary — прячет бинарник wireproxy внутрь исполняемого файла бота.

🪄 Автоматическая установка (Autostart)
В репозитории предусмотрен удобный .bat скрипт (install.bat) для скрытого развертывания бота на целевом ПК.
Соберите .exe файл, положите его рядом с install.bat и запустите батник. Скрипт сам спрячет бота в системную папку LocalAppData (под видом обновления браузера Edge) и пропишет в автозагрузку. Исходную папку после этого можно удалять.

🌐 Обход блокировок (DPI Bypass)
Бот оснащен автономной системой выживания в сети, которая работает по каскадному принципу:

* Режим Relayer (API Mirror): При запуске бот пытается связаться с вашим личным сервером Nginx (настраивается в переменных MIRROR_API и функции patched_create_connection), подменяя SNI и DNS-запросы.

* Режим WireGuard (Stealth): Если вы просто отправите боту в чат конфигурационный файл .conf от любого WireGuard сервера, бот автоматически:
Проверит его на работоспособность через изолированный SOCKS5 порт.
Надежно зашифрует и спрячет конфигурацию в системе (win_sys_net.json).
Жестко перенаправит весь свой трафик (и DNS) внутрь туннеля через переменные среды ОС.
При падении Зеркала бот автоматически запустит этот туннель при перезагрузке.

* Прямой доступ: Резервный канал к api.telegram.org, если блокировок в сети нет.
