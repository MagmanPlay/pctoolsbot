# 🖥️ PCTools: Telegram PC Controller

Мощный, скрытный и абсолютно автономный инструмент для удаленного управления компьютером (Windows) через Telegram бота. 

Отличительная черта проекта — **выживаемость в суровых сетяз**, высочайшая отказоустойчивость и работа **без прав администратора**. Бот способен пробивать жесткие фаерволы (DPI), проходить NTLM-авторизацию, обходить Captive-порталы и управлять системой даже из ограниченных учетных записей.

⚠️ **ДИСКЛЕЙМЕР:** Данный проект создан исключительно в образовательных целях и для личного использования (управления собственными ПК). Автор не несет ответственности за любой ущерб, причиненный использованием данного программного обеспечения. Использование скрытого управления на чужих ПК без согласия владельца незаконно.

## 🔥 Главные фишки

* **Enterprise Network Survival:** Бот умеет распознавать "заглушки" провайдеров (Captive Portals). Он не падает с ошибкой, а бесшумно уходит в спящий режим, ожидая, пока реальный пользователь не пройдет авторизацию в сети, после чего мгновенно выходит на связь.
* **Auto-NTLM & Smart Proxy:** Автоматический захват системных прокси Windows. Если корпоративный шлюз требует доменную авторизацию, бот сам "на лету" подгрузит нужные модули и пройдет аутентификацию.
* **Не требует прав Администратора:** 95% функционала (включая работу с файлами, скриншоты, управление окнами, веб-камеру и даже WireGuard) работает из-под обычного пользователя.
* **Встроенный WireGuard (User-Space):** Бот умеет поднимать изолированный WG-туннель прямо в оперативной памяти (через `wireproxy`), не создавая виртуальных сетевых адаптеров в системе.
* **Mass Build (Фабрика Ботов):** Встроенный скрипт многопоточной сборки для генерации десятков уникальных ботов за пару минут из одного файла с токенами.
* **Stealth Mode:** Абсолютная невидимость. Никаких мигающих черных окон. Логи бесшумно пишутся в системную папку `%TEMP%` под видом диагностики Windows.

## 🛠 Возможности

### 👁‍🗨 Наблюдение и Медиа
* **Оптимизированные скриншоты:** Быстрый скриншот экрана с точной отрисовкой позиции курсора. Сжатие "на лету" в легкий JPEG позволяет мгновенно передавать картинки даже через лимитированные Nginx-зеркала и медленный интернет.
* **Веб-камера:** Тихий захват фото. Встроена система *Auto-Repair* — если камера зависла, бот программно перезапустит драйвер без запроса админских прав.
* **Микрофон:** Фоновая запись звука (24kHz). Автоматическая нарезка и отправка файлов, если их размер превышает лимиты Telegram.

### 📂 Файлы и Окна (Window Manager)
* **Умный диспетчер задач:** Вместо бесконечного списка системного мусора, бот сканирует заголовки окон и показывает, **какие именно документы и вкладки** открыты у пользователя (например: `🔹 WINWORD.EXE 📄 Отчет.docx - Word`).
* Удобное скачивание файлов с ПК и загрузка файлов на ПК прямо через чат.
* Завершение (Taskkill) любых процессов.

### 🖱 Управление вводом и Плееры
* Эмуляция движений мыши и кликов.
* **Аппаратные горячие клавиши:** Имитация скан-кодов (Hardware Scan-Codes) для безотказного управления плеерами и YouTube: `Space` (Пауза/Пуск), `F` (Фулскрин видео), `F11` (Фулскрин браузера), `Esc`, `Alt+F4`, `Win+D`.
* **Менеджер ввода:** Жесткая блокировка физической мыши и клавиатуры (защита от детей/вмешательства), инверсия кнопок мыши.

### ⚙️ Дополнительно
* **Smart Update:** Бот умеет скачивать новую версию себя по прямой ссылке, заменять файл и надежно перезапускаться с помощью скрытой VBS-прослойки.
* Выполнение невидимых CMD-команд с возвратом вывода (stdout) в чат.
* Вывод системных `MessageBox` поверх всех окон (перехват фокуса).
* **Самоуничтожение:** Полное удаление скрипта из системы и автозагрузки по одной кнопке.

## 🚀 Установка и Настройка

### 1. Подготовка
Убедитесь, что у вас установлен **Python 3.8+**. Установите необходимые зависимости:
```bash
pip install pyTelegramBotAPI comtypes mouse opencv-python psutil sounddevice numpy scipy Pillow pySmartDL pynput pycaw requests urllib3 pysocks requests_negotiate_sspi
```
### 2. Настройка конфигурации
Откройте скрипт PCToolsBot.py (или ваше имя файла) и измените базовые настройки:

```Python
my_id = 123456789  # Ваш Telegram ID (бот будет игнорировать всех остальных)
bot_token = 'ВАШ_ТОКЕН_ОТ_BOTFATHER'
```
### 3. Компиляция в EXE (Одиночная)
Скачайте wireproxy.exe для Windows amd64 (приложена в релизах) и положите рядом со скриптом. Скомпилируйте бота:

```Bash
pyinstaller -w -F --hidden-import requests_negotiate_sspi --hidden-import pysocks --add-binary "wireproxy.exe;." PCToolsBot.py
```

-w — скрывает окно консоли.
-F — упаковывает всё в один .exe файл.
--hidden-import — принудительно зашивает библиотеки для пробива прокси.

### 🏭 4. Фабрика Ботов (Массовая сборка)
Если вам нужно скомпилировать ботов для нескольких ПК:

Создайте файл tokens.txt в формате ID:TOKEN:ALIAS (пример в релизе 1.1)
Положите рядом файл иконки icon.ico (по желанию).
Запустите скрипт сборщика: python builder.py.
Скрипт автоматически подменит токены, распараллелит сборку и аккуратно сложит готовые .exe файлы с вашей иконкой по отдельным папкам.

## 🪄 Автоматическая установка (Autostart)
В репозитории предусмотрен .bat скрипт (install.bat) для скрытого развертывания. Соберите .exe файл, положите его рядом с install.bat и запустите. Скрипт сам спрячет бота в LocalAppData и пропишет в автозагрузку.

## 🌐 Обход блокировок (Enterprise DPI Bypass)
Бот оснащен каскадной системой выживания, которая адаптируется под любую среду:
Спящий режим (Captive Portal Wait): При запуске бот проверяет сервера Apple/Firefox/Google. Если сеть перекрыта страницей авторизации провайдера, бот бесшумно ждет, пока пользователь не введет пароль в браузере.
Режим Relayer (API Mirror): Как только интернет появляется, бот захватывает системный прокси (если есть), проходит NTLM-авторизацию и стучится на ваше личное зеркало (Nginx). Трафик выглядит как обычный легитимный HTTPS-запрос (например, синхронизация облачного диска), что позволяет обходить фильтрацию DPI.
Режим WireGuard (Fallback): Если вы отправите боту .conf файл в чат, он сохранит его в зашифрованный кэш. При недоступности Зеркала бот автоматически поднимет изолированный SOCKS5-туннель через этот конфиг.
Прямой доступ: Резервный прямой канал к api.telegram.org для открытых сетей, где Telegram работает.
