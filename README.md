# 🖥️ PCTools: Telegram PC Controller

Мощный, скрытный и автономный инструмент для удаленного управления компьютером (Windows) через Telegram бота. 

Отличительная черта проекта — высочайшая отказоустойчивость сети и работа **без прав администратора**. Бот способен пробивать жесткие корпоративные фаерволы (DPI) и управлять системой даже из ограниченных учетных записей.

> ⚠️ **ДИСКЛЕЙМЕР:** Данный проект создан исключительно в образовательных целях и для личного использования (управления собственными ПК). Автор не несет ответственности за любой ущерб, причиненный использованием данного программного обеспечения. Использование скрытого управления на чужих ПК без согласия владельца незаконно.

---

## 🔥 Главные фишки

* **Не требует прав Администратора:** 95% функционала (включая работу с файлами, скриншоты, управление процессами и веб-камеру) работает из-под обычного пользователя.
* **Обход корпоративных блокировок (DPI Bypass):** Уникальная система `API Relayer` с перехватом DNS-запросов (Monkey Patching). Бот может работать в сетях, где полностью заблокирован Telegram и сторонние DNS-серверы.
* **Встроенный Proxy Manager:** Поддержка SOCKS5/HTTP прокси с возможностью переключения прямо через меню в Telegram.
* **Smart Update:** Бот умеет скачивать новую версию себя по прямой ссылке (`.exe`), заменять старый файл и перезапускаться "на лету" с помощью VBS-прослойки.

## 🛠 Возможности

### 👁‍🗨 Наблюдение и Медиа
* **Скриншоты:** Быстрый скриншот экрана или полноразмерный HD-снимок с точной отрисовкой позиции курсора.
* **Веб-камера:** Тихое создание фото. Встроена система *Auto-Repair* — если камера зависла или занята, бот попытается программно перезапустить драйвер захвата без запроса админских прав.
* **Микрофон:** Фоновая запись звука (24kHz). Автоматическая нарезка и отправка файлов, если их размер превышает лимиты Telegram (по 21 МБ).

### 📂 Файлы и Процессы
* Удобное скачивание файлов с ПК и загрузка файлов на ПК прямо через чат.
* Скачивание файлов на ПК по прямой URL-ссылке.
* Просмотр топ-25 процессов по потреблению **ОЗУ** (а не CPU).
* Завершение (Taskkill) любых процессов.

### 🖱 Управление вводом
* Эмуляция движений мыши, кликов и нажатий клавиш клавиатуры (`Alt+F4`, `Win+D`, `Enter` и др.).
* **Менеджер ввода:** Блокировка физической мыши и клавиатуры (отличная защита от детей или котов), инверсия кнопок мыши.

### ⚙️ Дополнительно
* **Выполнение CMD-команд:** Невидимое выполнение команд с возвратом вывода (`stdout`/`stderr`) прямо в чат, либо запуск видимого окна терминала.
* **Управление питанием:** Выключение и перезагрузка компьютера.
* **Отправка уведомлений:** Вывод системных `MessageBox` сообщений поверх всех окон.
* **Самоуничтожение:** Полное удаление скрипта из системы и автозагрузки по одной кнопке.

## 🚀 Установка и Настройка

### 1. Подготовка
Убедитесь, что у вас установлен **Python 3.8+**.
Установите необходимые зависимости:
```bash
pip install pyTelegramBotAPI comtypes mouse opencv-python psutil sounddevice numpy scipy Pillow pySmartDL pynput pycaw requests urllib3
```
2. Настройка конфигурации
Откройте скрипт PCToolsBot.py и измените базовые настройки на 183-184 строке в файле:

```Python
my_id = 123456789  # Ваш Telegram ID (бот будет игнорировать всех остальных)
bot_token = 'ВАШ_ТОКЕН_ОТ_BOTFATHER'
```
3. Компиляция в EXE (Опционально, но рекомендуется)
Чтобы запускать бота без консоли и привязки к Python на целевом ПК, скомпилируйте его:

```
pip install pyinstaller
pyinstaller -w -F PCToolsBot.py
```
Значения флагов:
-w — скрывает черное окно консоли (бот будет работать абсолютно незаметно в фоне).

-F — упаковывает все библиотеки и скрипт в один цельный .exe файл.

🪄 Автоматическая установка (Autostart)
В репозитории предусмотрен удобный .bat скрипт (install.bat) для автоматического развертывания бота на целевом ПК. Вам не нужно вручную прописывать пути в реестре или папке Startup.

Соберите .exe файл бота (как описано выше).

Положите готовый PCToolsBot.exe в одну папку рядом с файлом-установщиком install.bat.

Просто запустите .bat файл!

Скрипт сам перенесет бота в нужную системную директорию и автоматически пропишет его в автозагрузку Windows. После этого исходную папку с установщиком можно удалять — бот уже работает в фоне и будет автоматически запускаться при каждом включении компьютера.

🌐 Как работает API Mirror (DPI Bypass)?
Если целевой ПК находится в сети, где заблокирован доступ к api.telegram.org (например, школы, офисы), бот автоматически пытается перейти в режим Relayer.

Для этого нужен свой VPS-сервер с Nginx:

Поднимите Nginx на любом недорогом VPS (вне корпоративной/ограниченной сети).

Привяжите домен и настройте проксирование на https://api.telegram.org.

В скрипте в функции patched_create_connection укажите IP вашего сервера и домен:

```Python
if host == "youmirror.top":
    host = "mirrorIP"
```
В переменной MIRROR_API замените домен на свой:
```Python
MIRROR_API = "[https://youmirror.top:443/bot](https://youmirror.top:443/bot){0}/{1}"
```
Результат: Бот будет стучаться напрямую по IP-адресу сервера, игнорируя корпоративный DNS, но при этом передаст правильный Host заголовок и подменит SNI. Фаерволы увидят обычный защищенный трафик до вашего домена и пропустят его (в большинстве случаев).
